Apartment Management System Project

To run the project:

1. Download the zip file - "FinalProject_group3.zip"

2. Extract the file and copy the folder - "FinalProject_group3"

3. Paste the folder inside root directory (mamp/htdocs, xampp/htdocs, wamp/www, var/www/html)

4. Open phpmyadmin and create a database with the name "project_db"

5. Import "project_db.sql" file (included in the zipped file)

6. Run the script http://localhost:8888/FinalProject_group3/index.html (website home page)
